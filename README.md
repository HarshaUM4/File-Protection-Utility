# File Protection Utility

## Description
A Python program that protects file content using encoding and restores it when required.

## Features
- Protect file data
- Restore original file content
- Simple and easy to use
- Demonstrates file security concepts

## Technologies Used
- Python
- File Handling
- Base64 Encoding

## How to Run
1. Run the Python file.
2. Choose Protect or Restore.
3. Enter the file name.
4. Follow the on-screen instructions.
